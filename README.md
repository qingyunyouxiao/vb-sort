# 电子表格排序

### 如何过好每一天

1. 知道自己想做什么
2. 知道自己不想做什么
3. 去做你想做的事情，不去做你不想做的事情

### 示例数据

```
student_id	student_name	student_class	student_score
S001	张三	高一 (1) 班	85
S002	李四	高一 (2) 班	92
S003	王五	高一 (1) 班	78
S004	赵六	高一 (3) 班	95
S005	孙七	高一 (2) 班	88
S006	周八	高一 (3) 班	76
S007	吴九	高一 (1) 班	90
S008	郑十	高一 (2) 班	83
```

### 操作

1. 将表格类型保存为 xlsm 格式，新建表格，填入示例数据。
2. 打乱列的顺序，注意不要改变列数（默认为四列）。
3. 鼠标框选数据（包含标题），按下 Alt + F8 执行宏命令，可以看到数据完成了重新排列。
